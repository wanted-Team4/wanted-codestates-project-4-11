### 📊 Getting Start

1. `clone` the repository

```
$ git clone "https://github.com/wanted-Team4/wanted-codestates-project-4-11"
```

2. `Install` dependencies

```
$ npm install
```

3. `start` the project

```
$ npm run serve
```

4. `Setting` prettier

```
$ npx prettier --write .
```
